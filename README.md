# Accra Real Estate Demo Website

A modern, mobile-first real estate website demo for Accra-based real estate agents, optimized to convert visitors into WhatsApp inquiries.

## Features

- Modern, professional design
- Mobile-first responsive layout
- WhatsApp integration for easy communication
- Trust-focused UI/UX
- Smooth animations with Framer Motion
- Production-ready code structure

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

### Update Business Information

Edit the constants in `app/page.tsx`:
- `BUSINESS_NAME`: Your business name
- `WHATSAPP_NUMBER`: Your WhatsApp number (format: 233XXXXXXXXX)
- `LOCATION`: Your business location

### Update Properties

Edit the `properties` array in `app/page.tsx` to add/remove property listings.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
